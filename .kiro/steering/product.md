# Product Overview

StreamViewer is a Python package for real-time data visualization, primarily designed for monitoring and visualizing streaming data from Lab Streaming Layer (LSL) streams.

## Core Purpose

- Real-time visualization of streaming data, especially neural/EEG data
- Modular architecture with pluggable data sources and visualization renderers
- Pre-built GUI applications for LSL stream monitoring

## Key Applications

- `lsl_viewer`: Main GUI application for monitoring LSL streams with multiple visualization options
- `lsl_status`: Stream status monitoring
- `lsl_switching_viewer`: Switching between different stream views
- `lsl_viewer_custom`: Custom stream visualization with command-line configuration

## Target Users

Researchers and developers working with real-time streaming data, particularly in neuroscience and EEG applications using the Lab Streaming Layer protocol.
